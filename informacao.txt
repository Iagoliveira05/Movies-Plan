Criar um aplicativo para organizar filmes e séries que deseja assistir posteriormente

Opções:
    • Adicionar filmes/séries como desejos
    • Excluir filmes/séries
    • Marcar como visto, assim sumirá dos desejos e irá para os vistos
    • Favoritar filmes ou séries
    • Sortear um filme ou série

BANCO DE DADOS

Utilizar SQLite3 para criar o banco de dados
